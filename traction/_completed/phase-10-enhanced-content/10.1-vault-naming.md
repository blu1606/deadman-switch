# 10.1 Vault Naming

> Cho phép đặt tên vault để dễ quản lý.

## Goal
User có thể đặt tên cho vault (VD: "Crypto Keys", "Letter to Mom") thay vì chỉ thấy địa chỉ.

## Specs
- Max: **32 ký tự**
- Optional khi tạo vault
- Có thể đổi tên sau qua Edit Settings

## Contract Changes
1. Thêm field `name: String` vào Vault struct
2. Cập nhật SPACE (+36 bytes)
3. `initialize_vault` nhận thêm param `name`
4. `update_vault` nhận thêm param `new_name: Option<String>`
5. Thêm error `NameTooLong`

## Frontend Changes
- Thêm input tên ở bước tạo vault
- Dashboard hiển thị tên thay vì địa chỉ
- EditVaultModal cho phép đổi tên

## Migration
Vault cũ sẽ có `name = ""`. Frontend fallback: hiển thị địa chỉ rút gọn nếu không có tên.
